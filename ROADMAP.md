# Roadmap : Mon Assistant Scolaire 🚀

Ce document présente la feuille de route pour le développement futur de l'application "Mon Assistant Scolaire".

## 🌟 Phase 1 : Fondation (TERMINÉE)
- [x] **Assistant IA** : Chatbot éducatif sécurisé avec Gemini.
- [x] **Activités variées** : Quiz, Histoires, Dictionnaire, Calcul Mental, Anecdotes.
- [x] **Gamification basique** : Système d'étoiles/points pour récompenser l'apprentissage.
- [x] **Interface Enfant** : UI colorée, simple et responsive (Mobile & Desktop).
- [x] **Historique cloud** : Migration vers Supabase pour la persistance et la synchro.

## 🚀 Phase 2 : Personnalisation & Suivi (TERMINÉE)
- [x] **Profils Utilisateurs** : Création de profils (Avatar, Prénom, Classe/Niveau).
- [x] **Multi-enfants** : Gestion de plusieurs enfants par compte parent.
- [x] **Tableau de bord de progression** : Graphiques montrant les scores par matière.
- [x] **Défis quotidiens thématiques** : Mot et problème du jour avec choix du thème (Espace, Nature, etc.).
- [x] **Niveaux de difficulté** : Adaptation automatique du contenu selon le niveau (CP à CM2).

## 🎨 Phase 3 : Pédagogie Avancée & Multimodalité (EN COURS)
- [x] **Atelier "Explique avec tes mots"** : Vérification de la compréhension réelle via reformulation.
- [x] **Révision Espacée (SRS)** : Système intelligent de rappel pour la mémorisation long terme.
- [x] **Diagnostic IA & Remédiation** : Détection des lacunes et conseils personnalisés.
- [/] **Reconnaissance Vocale** : Pose de questions à l'oral (Speech-to-Text).
- [ ] **Synthèse Vocale (TTS)** : Lecture des réponses par l'IA (Aide aux petits lecteurs).
- [x] **Aide aux devoirs par l'image** : Analyse de photos d'exercices (Vision).
- [x] **Dessin interactif** : Zone de dessin pour expliquer ou s'exprimer.

## 🛡️ Phase 4 : Écosystème Parent & Sécurité
- [x] **Espace Parents Premium** : Dashboard complet avec code PIN.
- [x] **Contrôle du Temps** : Limites d'utilisation quotidienne et couvre-feu.
- [x] **Plan Hebdo Parent-Enfant** : Objectifs partagés et guides d'accompagnement.
- [ ] **Export PDF Intelligent** : Fiches de révision et cahiers de vacances personnalisés.
- [ ] **Notifications Intelligentes** : Rappels sur mobile pour le parent (progrès, alertes lacunes).

## 🔥 NOUVELLE VISION : Fonctionnalités à venir (+25 innovations)

### 🧩 Gamification & Motivation
1.  **Boutique d'Avatars Rare** : Utiliser ses points pour débloquer des accessoires et des animations uniques.
2.  **Système de Mentors** : Les enfants de CM2 peuvent parrainer/aider des enfants de CP (quiz simplifiés).
3.  **Duels en Temps Réel** : Défier un ami (sécurisé) sur un quiz de rapidité.
4.  **Pass d'Apprentissage** : Chemins de progression saisonniers avec récompense finale thématique.
5.  **Banque de points "Argent de poche"** : Convertir (virtuellement) ses succès en temps d'écran bonus ou récompenses réelles.

### 🧠 Intelligence Artificielle & Outils
6.  **Analyse d'écriture manuscrite** : Photographier son cahier pour que l'IA corrige la formation des lettres (CP/CE1).
7.  **IA Émotionnelle** : Détecter les signes de frustration via le ton de voix ou l'usage pour proposer une pause.
8.  **Générateur de Flashcards** : Création automatique de cartes mémo à partir des erreurs passées.
9.  **Podcast du Jour** : Résumé audio de 2 minutes des 3 choses apprises dans la journée.
10. **Wikipedia Junior intégré** : Source d'information certifiée vulgarisée par l'IA.
11. **Détecteur de Plagiat Éducatif** : Apprendre à l'enfant à ne pas copier-coller mais à reformuler.

### 🏠 Vie Quotidienne & Collaboration
12. **Mode "Tableau Blanc Familial"** : Espace de dessin collaboratif synchrone entre parent et enfant.
13. **Suivi du Sommeil vs Performance** : Analyse croisée entre heures d'étude et qualité du repos.
14. **Calendrier Scolaire IA** : Intégrer les dates d'examens réelles pour adapter le plan de révision.
15. **Mode "Mission Commando Dodo"** : Guide de relaxation/méditation par l'assistant avant le coucher.
16. **Scripts de Discussion Parentale** : L'IA suggère des questions à poser au dîner pour approfondir un sujet vu.

### 🌍 Accessibilité & Inclusion
17. **Mode "Dyslexie"** : Police OpenDyslexic et contrastes spécifiques.
18. **Traducteur Visuel AR** : Pointer un objet réel pour avoir son nom et sa définition en 3 langues.
19. **Support Multi-langues** : Apprendre l'anglais ou l'espagnol via des conversations immersives.
20. **Mode "Sans Écran" (Print & Play)** : Générer des labyrinthes et sudokus basés sur des notions de cours.

### 🛠️ Technique & Social (Sécurisé)
21. **Mode Hors-ligne (PWA)** : Accès aux jeux et révisions sans connexion internet.
22. **Profil "Classe"** : Permettre à un instituteur de voir les statistiques globales de ses élèves.
23. **Cloud Backup Parental** : Export complet des données pour archivage des travaux de l'enfant.
24. **IA de Détection de Cyber-harcèlement** : Module de sensibilisation intégré aux discussions.
25. **API Partenaires** : Intégration de contenus issus de musées ou d'éditeurs scolaires.
